4. jekyll theme
   
    Make sure to specify in `_config.yml`:
    
        theme: jekyll-theme-architect

    and check that the gem is specified in `Gemfile`.

        gem "jekyll-theme-architect", "~> 0.2.0"

    If the gem has not been installed, run 
        
        $ bundle install
